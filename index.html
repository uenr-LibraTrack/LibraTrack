How to Use UENR LibraTrack

  1. Check in: Go to the Check In page, enter the Library Code (e.g. LIB-MAIN) and your Student ID.
  2. Check out: Enter the exact same Student ID on the Check In page to release your seat.
  3. View Occupancy: Live seat counts and percentages are visible on the main dashboard.
  4. Admin Codes: Admins can print library entrance codes from the Admin Panel.
  5. Get Alerts: Click "Enable Desktop Alerts" on the Notifications page.
